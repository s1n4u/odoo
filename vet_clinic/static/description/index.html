Veterinary Clinic Module

Overview

This module provides a full-featured management system for veterinary clinics, including functionality for managing patients, doctors, appointments, diagnoses, medicines, and diseases.

Features

  • Animal patients with species, breed, and birthday tracking
  • Owner linking via res.partner
  • Doctor management using linked users
  • Appointments with status tracking and quick wizard
  • Diagnosis records with treatments, medicines, and recommendations
  • Birthday email reminders and discount logic
  • Customizable species and breeds
  • Integrated QWeb report for diagnoses

Technical

  • Compatible with Odoo 17+
  • Extensive use of mail.thread for chatter
  • Fully translatable (EN/UA)
  • Demo data provided for testing
