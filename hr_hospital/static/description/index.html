Модуль "Управління лікарнею" (hr_hospital)

Модуль для автоматизації роботи лікарні в Odoo. Дозволяє вести облік лікарів, пацієнтів, візитів, діагнозів, хвороб, спеціальностей, а також містить майстри для звітів і масових дій.

Основні можливості

  • Облік лікарів, пацієнтів, інтернів, менеджерів та адміністраторів
  • Гнучка система прав доступу
  • Візити, діагнози, класифікатор хвороб
  • PDF/HTML звіти по лікарях і діагнозах
  • Майстри для масового призначення лікаря та формування звітів
  • Підтримка української мови
  • Демо-дані для швидкого старту

Структура ролей

  • Пацієнт — перегляд лише своїх візитів
  • Інтерн — перегляд і редагування своїх візитів
  • Лікар — перегляд і редагування своїх та візитів своїх інтернів
  • Менеджер — перегляд усіх візитів
  • Адміністратор — повний доступ, включаючи видалення

Module "Hospital Management" (hr_hospital)

This module automates hospital management in Odoo. It supports doctors, patients, visits, diagnoses, disease classification, and includes wizards for reporting and batch actions. Full Ukrainian localization and demo data included.
